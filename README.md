# STUDENT-RESULT-ANALYZER
A menu-driven Python mini project developed to manage and analyze student academic records. The application allows users to add, search, update, and delete student records while performing result analysis such as grade calculation, class average, topper identification, identifies lowest scorer, and pass/fail reporting.

## Features
- Add Student
- Search Student
- Update Student
- View Student Details
- Delete Student
- Calculate Grades
- Find Class Topper
- Find Lowest Scorer
- Calculate Class Average
- Pass/Fail Report

## Technologies Used
- Python

## Concepts Used
- Functions
- Conditional Statements
- Loops
- Lists
- Dictionaries
- String Handling
- User Input & Output

This project was built as part of my Python learning journey while pursuing Data Analytics. It helped me strengthen my understanding of Python fundamentals and improve my problem-solving skills by applying them to a real-world scenario.
